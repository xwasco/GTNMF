GTNMF

Context-aware non negative matrix factorization clustering

View project on GitHub

What is ?

GTNMF is a game-theoretic method to refine the clustering obtained via NMF. The refinement of the assignment is obtained throuh this game-theoretic setting:

  • The elements to be clustered are the players
  • The similarities between the players defines who is playing with. The rationale is that players want to play with similar level players.
  • The assignment elements-clusters takes into account the similarity between objects and their membership choices rearraging the un-coherent ones.

The initialization of the game is provided by the soft-clustering obtained by an NMF clustering method.
